- docker-compose.yml
- .env.example
- scripts for seeding / development notes

## How to use
1. Extract the zip or clone the repo.
2. Copy `.env.example` to `.env` and fill values.
3. Use Docker Compose to bring up services or run frontend/backend separately.

### Quick (local dev)
```bash
# from repo root
docker-compose up --build
```
- Frontend: http://localhost:3000
- Backend:  http://localhost:4000
